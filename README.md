# highway-to-hack

Simulation scripts with SUMO and TraCI to quantify impact of scenarios based in the security issues found in ETSI ITS standards.

Part of a paper published in Computer Standards & Interfaces:

- Roberto Gesteira-Miñarro, Takahito Yoshizawa, and Rafael Palacios and Gregorio López. "Highway to Hack - Security gaps in ETSI ITS standards". _Computer Standards & Interfaces_. vol. 97, pp. 104133-1 - 104133-10 (2026). issn: 0920-5489. doi: [10.1016/j.csi.2026.104133](https://doi.org/10.1016/j.csi.2026.104133).
